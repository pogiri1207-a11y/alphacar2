## 요구사항 우선순위 가이드

### 우선순위 정의
- **P0 (Critical)**: 필수 기능, 서비스 핵심 기능
- **P1 (High)**: 중요 기능, 사용자 경험에 큰 영향
- **P2 (Medium)**: 일반 기능, 추가 편의 기능
- **P3 (Low)**: 선택 기능, 향후 개선 사항

### 주요 기능 우선순위 예시
- **P0**: 로그인, 견적 비교, 차량 검색, 찜하기
- **P1**: 커뮤니티, 마이페이지, 상담신청
- **P2**: 이벤트, 혜택 안내, 우주 게임
- **P3**: 통계 기능, 관리자 기능

## 비고

### 개발 환경
- 개발 서버: `http://localhost:3000`
- 백엔드 서버: `http://192.168.0.160:{PORT}`

### 기술 스택
- Frontend: Next.js 16.0.5, React 19.2.0
- Language: TypeScript / JavaScript
- Styling: Inline Styles (CSS-in-JS)

### 인증 방식
- JWT 토큰 인증
- 소셜 로그인 (카카오, 구글)
