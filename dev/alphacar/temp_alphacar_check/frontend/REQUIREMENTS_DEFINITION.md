## 우선순위 기준

### 높음 (High Priority)
- 핵심 기능으로 서비스 운영에 필수적인 기능
- 사용자 경험에 직접적인 영향을 미치는 기능
- 예: 회원가입, 로그인, 견적 비교, 차량 검색

### 중간 (Medium Priority)
- 서비스 품질 향상을 위한 기능
- 사용자 편의성을 높이는 기능
- 예: 찜하기, 커뮤니티, 이벤트

### 낮음 (Low Priority)
- 부가 기능 또는 향후 확장 기능
- 서비스 운영에 필수적이지 않은 기능
- 예: 게임, 일부 통계 기능

## 기능별 상세 요구사항

### 1. 사용자 인증 및 회원 관리

#### 1.1 회원가입
- **이메일 회원가입**
  - 필수 입력 항목: 이메일, 비밀번호, 이름, 전화번호
  - 이메일 중복 체크
  - 비밀번호 정책: 최소 8자 이상, 영문/숫자/특수문자 조합
  - 이메일 인증 필수
  - 약관 동의 (필수/선택)

- **SNS 회원가입**
  - 카카오/구글 OAuth 연동
  - 소셜 계정 정보로 자동 회원가입
  - 추가 정보 입력 (이름, 전화번호 등)

#### 1.2 로그인
- **이메일 로그인**
  - 이메일/비밀번호 입력
  - 로그인 실패 시 에러 메시지 표시
  - 로그인 성공 시 JWT 토큰 발급

- **SNS 로그인**
  - 카카오/구글 OAuth 인증
  - 인증 성공 시 자동 로그인
  - 소셜 계정 정보로 사용자 정보 자동 연동

- **자동 로그인**
  - 체크박스 선택 시 토큰을 localStorage에 저장
  - 다음 접속 시 자동 로그인 처리

### 2. 견적 비교

#### 2.1 비교견적
- 최대 3대까지 동시 비교 가능
- 차량 선택 후 견적 비교 요청
- 비교 결과: 가격, 옵션, 제원을 테이블 형식으로 표시
- 비교견적 저장 기능

#### 2.2 개별견적
- 제조사 → 모델 → 트림 → 옵션 순서로 선택
- 각 단계별 가격 및 제원 정보 표시
- 최종 견적 가격 계산 및 표시
- 딜러 정보 포함
- 개별견적 저장 기능

### 3. 차량 검색

#### 3.1 검색 기능
- 키워드 검색 (차량명, 모델명)
- 브랜드별 필터링
- 검색 결과: 이미지, 가격, 제원 정보 표시
- 검색 결과에서 바로 찜하기, 비교하기, 견적 요청 가능

#### 3.2 차량 상세
- 차량 이미지 갤러리
- 상세 제원 정보
- 옵션 목록 및 가격
- 찜하기 버튼
- 견적 요청 버튼

### 4. 마이페이지

#### 4.1 견적함
- 저장된 견적 목록 표시
- 견적 개수 표시
- 견적 상세 조회
- 견적 삭제

#### 4.2 포인트
- 현재 보유 포인트 표시
- 포인트 적립/사용 내역
- 포인트 적립 방법 안내

#### 4.3 찜한 차량
- 찜한 차량 목록 그리드 표시
- 차량 상세 모달
- 찜 해제 기능

#### 4.4 최근 본 차량
- 최근 조회한 차량 목록 표시
- 차량 클릭 시 개별견적 페이지로 이동
- 목록 삭제 기능

### 5. 커뮤니티

#### 5.1 게시글
- 카테고리별 필터링 (전체/구매고민/오너리뷰)
- 게시글 목록 (제목, 작성자, 작성일, 조회수)
- 게시글 상세 (내용, 댓글)
- 게시글 작성/수정/삭제
- 검색 기능
- 페이지네이션 (10개씩)

#### 5.2 댓글
- 댓글 작성/수정/삭제
- 댓글 목록 표시

### 6. 이벤트

#### 6.1 이벤트 조회
- 진행중/종료된 이벤트 탭 전환
- 이벤트 카드 형식으로 표시
- 이벤트 상세 페이지
- 이벤트 참여 기능

### 7. 관리자

#### 7.1 사용자 관리
- 사용자 목록 조회 및 검색
- 사용자 상세 정보 확인
- 사용자 차단 기능

#### 7.2 게시글 관리
- 공지사항 작성/수정/삭제
- 커뮤니티 게시글 관리
- 부적절한 게시글 삭제

#### 7.3 통계
- 사용자 통계 (가입자 수, 활성 사용자)
- 견적 통계 (견적 요청 수, 인기 차량)
- 게시글 통계

## 비기능 요구사항

### 성능
- 페이지 로딩 시간: 3초 이내
- API 응답 시간: 1초 이내
- 이미지 최적화: WebP 형식 지원

- HTTPS 통신 필수
- JWT 토큰 기반 인증
- SQL Injection 방지
- XSS 공격 방지
- CSRF 토큰 사용

### 호환성
- 브라우저: Chrome, Safari, Firefox, Edge 최신 버전
- 모바일: iOS 12+, Android 8+
- 반응형 디자인 지원

### 접근성
- 키보드 네비게이션 지원
- 스크린 리더 호환
- 색상 대비 준수
